ビルド方法（実行ファイルの作り方）

先ほどクローンしたディレクトリには、docker環境に関する設定ファイルが含まれている。もしJavaの開発環境が整っていなければ、Dockerを使ってビルドを行うことができる。(Javaの開発環境が整っている場合、1, 2はスキップして良い)

  1. 以下のコマンドでDockerイメージをビルドする。 
    $ docker-compose build
  2. ビルドが成功したら、以下のコマンドでコンテナを起動する。 
    $ docker-compose run java bash
  3. 以下のコマンドを実行して、アプリケーションをビルドする。 
    $ make app
    env LC_ALL=ja_JP.UTF-8 ant install
